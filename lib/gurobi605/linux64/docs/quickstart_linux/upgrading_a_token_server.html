Upgrading a token server

If you want to upgrade a token server from an earlier version of the Gurobi Optimizer, you will need to perform the following steps (on the machine running the token server):

  1. Stop the old token server.
  2. Install the new version of the Gurobi Optimizer.
  3. Upgrade your license file (or modify GRB_LICENSE_FILE to point to the new license file).
  4. Start the new token server.
