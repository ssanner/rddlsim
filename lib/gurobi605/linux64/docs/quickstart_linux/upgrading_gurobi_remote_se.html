Upgrading Gurobi Remote Services

If you want to upgrade Gurobi Remote Services from an earlier version of the Gurobi Optimizer, you will need to perform the following steps (on the machine running Gurobi Remote Services):

  1. Stop the old Gurobi Remote Services.
  2. Install the new version of the Gurobi Optimizer.
  3. Upgrade your license file (or modify GRB_LICENSE_FILE to point to the new license file).
  4. Start the new Gurobi Remote Services.
