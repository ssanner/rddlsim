Results

Once we've added the model constraints, we call optimize and then output the optimal solution: 

if m.status == GRB.status.OPTIMAL:
  for h in commodities:
    print '\nOptimal flows for', h, ':'
    for i,j in arcs:
      if flow[h,i,j].x > 0:
        print i, '->', j, ':', flow[h,i,j].x
