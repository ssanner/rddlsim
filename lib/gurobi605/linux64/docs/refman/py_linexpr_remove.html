LinExpr.remove()

remove ( item )

Remove a term from a linear expression.

Arguments:

item: If item is an integer, then the term stored at index item of the expression is removed. If item is a Var, then all terms that involve item are removed.

Example usage:

  e = x + 2 * y + 3
  e.remove(x)
