QuadExpr.getVar1()

getVar1 ( i )

Retrieve the first variable for a single quadratic term of the quadratic expression.

Return value:

First variable associated with the quadratic term at index i in the quadratic expression.

Example usage:

  e = x * x + 2 * y * y + z
  print e.getVar1(1)
