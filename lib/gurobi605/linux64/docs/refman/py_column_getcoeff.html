Column.getCoeff()

getCoeff ( i )

Retrieve the coefficient from a single term in the column.

Return value:

Coefficient for the term at index i in the column.

Example usage:

  col = Column([1.0, 2.0], [c0, c1])
  print col.getCoeff(1)
